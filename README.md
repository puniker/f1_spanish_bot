# F1 
Bot that returns information about the current season by Telegram. the data is obtained from the [Api Sports](https://api-sports.io)


# Set up development environment

```bash
npm install
npm run dev
```

## Environment varibales

- RAPID_API_KEY 
- TELEGRAM_BOT_API_KEY
